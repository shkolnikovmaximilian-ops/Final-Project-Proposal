# Re-Usable Blender Rig UI

## Repository
https://github.com/shkolnikovmaximilian-ops/Final-Project-Proposal.git

## Description
This program will create an easy-to-understand and re-usable rig UI to make animation easier for animators

## Features
- Feature 1 - Bone Visibility toggle (ON/OFF)
	- At the click of a button, you will be able to toggle the visibility of certain bone groups
- Feature 2 - Character Mesh Visibility (ARMS/LEGS/TORSO)
	- At the click of a button, you will be able to toggle the visibility of certain parts of the character mesh
- Feature 3 - IK to FK and FK to IK switch
	- At the click of a button, you will be able to snap an IK bone to the position of its corresponding FK bone, and vise versa
- Feature 4 - Color Wheel
	- At the click of a button, you will be able to pull up a color wheel to change the color of the entire character

## Challenges
- I will need to learn how to code python in blender, and mainly learn how to create new panels in blender's user interface
- I will need to learn how to put layer masks or bone groups into the ui python script
- I will need to learn how to make an IK to FK switch mechanism in blender python

## Outcomes
Ideal Outcome:
- The Ideal Outcome is to have a fully functioning ui that streamlines the process of animating with an easy to use character rig 

Minimal Viable Outcome:
- The Minimal outcome is to at least be able to toggle the visibility of certain bone groups to make the rig look less messy
