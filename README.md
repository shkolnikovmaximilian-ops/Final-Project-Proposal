## Project Description
This final project showcases a well crafted blender script coded fully in python. When used, the script will take advantage of blender's grease pencil functions to create a simple and interactable user interface that is linked directly to your chosen blender rig. You will be able to hide any bone group you want, hide any body part you want, change the color of your character, switch any fk bone to an ik bone (and vice versa), and change specific properties of any bone listed in the rig UI. And the best part is, this rig is re-usable accross any other rig with a similar bone structure
